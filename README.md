# Student Management System

A basic web-based **Student Management System** built using **Flask** and **SQLite**.  
This project allows managing student records with simple create, update, view, and delete operations.

The focus of this project is on understanding **backend fundamentals**, database interaction, and request handling using Flask.

---

## Features

- Add student details
- View list of students
- Edit student information
- Delete student records
- Data stored persistently using SQLite
- Clean and simple user interface

---

## Tech Stack

- **Backend:** Python, Flask  
- **Database:** SQLite  
- **Frontend:** HTML, CSS
